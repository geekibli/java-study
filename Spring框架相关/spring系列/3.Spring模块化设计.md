# 3.Spring模块化设计


Spring的每个模块都做了详细的拆分，可以做到按需引用。Spring拆分的模块大致有20+。

可以看下Github上面[源码](https://github.com/spring-projects/spring-framework)的结构：

![](https://oscimg.oschina.net/oscnet/up-edb33a19efce10fcb573a84ac8d106fb88b.png)




- spring-aop 
- spring-aspects 
- spring-context 
- spring-core 
- spring-expression ：spring表达式
- spring-beans
- spring-context-indexer 
- spring-context-support 
- spring-instrument 
- spring-jcl ：统一日志管理
- spring-jdbc
- spring-jms ： 消息服务
- spring-messaging ：消息的统一实现标准
- spring-orm ：hibernate
- spring-oxm ：xml编列
- spring-test 
- spring-tx ：事务抽象，重点难点
- spring-web 
- spring-webflux 
- spring-webmvc 
- spring-websocket
