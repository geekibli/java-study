### 搜索可用镜像
`docker search tutorial`

### 检查运行的镜像
`docker inspect efe`

### 发布镜像
`docker push`

### 下载镜像
`docker pull`

### 在容器中安装新的程序 
`apt-get update`
`apt-get install vim`

### 保存对容器的修改 
`docker commit pid`
